Work In Progress :
An interface for real estate investors in Paris

This web site will allow investors to generate automated business plan for appartment in Paris

This project is divided into three parts

  • Collecting real time and historical data from common real estate websites seloger.com; loueragile; etc...
    For collecting data, we will use both web scrapping and common real estate API. All scripts are being developped in python. Librairies used :
    #data analysis: pandas,numpy;
    #scrapping : requests,BeautifulSoup,re, os
    #storing : pymongo & dns
    The scripts will be scheduled on Jenkins and run on a PiServer.
  • Constructing a local database:
    For scalling purposes as well as personal interest... the database will be MongoDB, and will be storing (not exhaustivelly):
    • Online real estate offers data: price, location, number of rooms, etc..
    • External parameters influencing the price of an appartment/house :
    • Feedbacks from the website : ratings, site usage, etc..
    • historical market data : price trends for Paris' 'arrondissements', other relevent historical data.
  • Developping and designing the website:
    The website is being developped with npm/NodeJS & React.
    Server interface was developped with Express
