Work In Progress :
An interface for real estate investors in Paris

This web site will allow investors to generate automated buisness plan for appartment in Paris

This project is divided into three parts

  • Collecting real time data from common real estate websites seloger.com; loueragile; etc... , collecting historical data
    For collecting data, we will use both web scrapping and common real estate API for real estate. All scripts are being developped in python. The scripts will be scheduled on Jenkins and run on a PiServer.
  • Constructing a local database:
    For scalling purposes as well as personal interest... the database will be MongoDB, and will be storing (not exhaustivelly):
    • Online real estate offers data: price, location, number of rooms, etc..
    • External parameters influencing the price of an appartment/house :
    • Feedbacks from the website : ratings, site usage, etc..
    • historical market data : price trends for Paris' 'arrondissements', other relevent historical data.
  • Developping and designing the website:
    The website is being developped in React.
